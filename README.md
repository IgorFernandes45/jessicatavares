# Landing page elegante inspirada no estilo do template Webnode

## Como abrir no VS Code
1. Extraia o `.zip`
2. Abra a pasta no VS Code
3. Rode:

```bash
npm install
npm run dev
```

## Onde personalizar
Edite principalmente:

```bash
lib/site-config.ts
```

Lá você troca:
- nome da marca
- textos
- botões
- serviços
- endereço
- mapa
- WhatsApp
- Instagram
- email
- imagens

## Estrutura
- `app/page.tsx` → layout da landing
- `components/reveal.tsx` → efeito ao rolar
- `components/whatsapp-float.tsx` → botão flutuante
- `lib/site-config.ts` → conteúdo editável

## Observação
O projeto foi criado **inspirado no estilo visual** do modelo enviado, com estrutura própria em Next.js para você personalizar livremente.
